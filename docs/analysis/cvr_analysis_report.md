# 🎯 셀러별 주문 전환율(CVR) 분석 보고서

## 1. 개요
본 보고서는 제주 농산물 판매 데이터와 클릭 데이터를 결합하여 셀러별 마케팅 효율성을 측정하는 주문 전환율(CVR) 분석 결과를 담고 있습니다.

## 2. 주요 분석 지표
- **CVR (Conversion Rate)**: `(주문 건수 / 상품 클릭 수) * 100`
- **분석 대상**: 최신 `salesclick_1.csv` 및 `preprocessed_data_1.csv` 기준 상위 셀러

## 3. 분석 결과 요약

### 고효율 셀러 (High CVR)
- **주요 특징**: 낮은 클릭수에도 불구하고 높은 주문 전환을 기록. 상품 상세 페이지의 설득력이 높거나, 타겟팅된 유입이 효과적임.
- **전략적 제언**: 유입량(클릭수) 확대를 위한 광고 노출 증대 권장.

### 저효율 셀러 (Low CVR)
- **주요 특징**: 많은 클릭수가 발생하나 실제 주문으로 이어지는 비율이 낮음. 가격 경쟁력 부족 또는 상세 페이지 이탈률이 높을 것으로 추정.
- **전략적 제언**: 상세 페이지 UI/UX 개선 및 가격 소구점 재수립 필요.

## 4. 대시보드 반영 현황
- **위치**: `👥 고객 분석 및 재구매` 탭 하단
- **기능**: 선택된 필터(기간, 품종 등)에 따라 실시간으로 셀러별 CVR 순위 시각화 

## 5. 결론 및 향후 계획
CVR 지표를 통해 단순 매출액 이상의 '마케팅 효율' 관점에서 셀러를 평가할 수 있게 되었습니다. 향후 광고비(Ad Spend) 데이터를 확보할 경우 ROAS(광고비 대비 매출액) 분석으로 고도화가 가능합니다.
